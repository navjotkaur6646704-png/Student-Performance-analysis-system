# 🎓 Student Performance Analysis System

A Python-based Student Performance Analysis System that manages student records, analyzes academic performance, generates reports, visualizes data, and predicts student performance using Machine Learning.

---

## 🚀 Features

- Add, Update, Delete & Search Students
- Student Ranking
- Percentage & Grade Calculation
- CGPA Calculation
- Pass/Fail Analysis
- Subject-wise Highest & Average Marks
- Attendance Analysis
- Excel & CSV Report Export
- Interactive Streamlit Dashboard
- Machine Learning Prediction

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy

- Streamlit

## ▶️ Run the Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Dashboard:

```bash
python -m streamlit run dashboard.py
```

Run Console Application:

```bash
python main.py
```
